4/20–4/26 週銷售分析

本週總營收 NT$292,720，較上週（4/13–4/19，NT$274,540）回升約 6.6%，件數從 330 件增至 331 件。整體溫和回升，週末（4/25–4/26）衝刺明顯，兩日合計佔全週近 47%。

📌 品牌系列（NT$63,160）
品牌系列本週大幅反彈，較上週 NT$48,240 成長 31%。yufu（NT$32,080 / 25件）與 Mitty girls（NT$23,400 / 32件）並列主力，Babyboomer 本週有感回溫（NT$5,880 / 7件），Masako 仍偏低（NT$1,800 / 1件）。建議持續觀察 Babyboomer 是否進入復甦，可考慮小量補貨測試市場。

📌 特選新品（NT$61,150）
特選新品穩定成長，較上週 NT$59,550 微升 2.7%。褲款仍是絕對主力（NT$46,300 / 42件），墨鏡本週明顯回升（NT$9,600 / 24件），氣溫升高帶動需求。腰鏈褲鏈本週掛零，建議確認庫存狀況。

📌 二手古著（NT$168,410）
古著本週較上週（NT$166,750）持平微升 1%，表現穩健。Polo 襯衫本週大幅回彈（NT$37,300 / 29件，上週 NT$17,500），是本週最大驚喜。格紋襯衫持續強勢（NT$21,100 / 25件），素色襯衫本週異軍突起（NT$10,250 / 9件）。條紋襯衫本週明顯降溫，建議調整陳列位置。領帶穩定輸出（NT$7,200 / 16件），客群持續鞏固中。
